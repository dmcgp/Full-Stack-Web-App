# Contributing

## Workflow
- Fork and create feature branches
- Write tests and update docs
- Run lint and tests locally
- Open PR with a clear description

## Setup
- Backend: Java 17, `./mvnw` commands
- Frontend: Node 18+, `npm install`
- DB: `docker compose -f ./infra/docker-compose.yml up -d`

## Standards
- Conventional commits
- Meaningful PRs and issues
- Code style via ESLint/Prettier and EditorConfig
